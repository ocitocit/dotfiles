# Usage
place this files on  ~/.config/nvim

## Require
[Ripgrep](https://github.com/BurntSushi/ripgrep)

make dir if you don't have that 
````
mkdir .config
cd ~/.config
mkdir nvim
cd nvim
````
place it the files on nvim dir.
